<h1>一、火灾信号探测原理</h1>
火灾以接触式（物质流）和非接触式（能量流）的形式向外释放能量。接触式形式包括可燃气体、燃烧气体和烟雾、气溶胶等；非接触式如声音、辐射等。火灾探测技术就是试图从火灾过程中提取适当的特征作为探测识别火灾的参数，利用敏感元件将火灾中出现的物理化学特征转换为另外一种易于处理的物理量。

火灾探测算法分为接触式火灾探测算法和非接触式火灾探测算法两类。其中接触式火灾探测算法主要应用于感温、感烟和气体传感器中。非接触式则主要应用于图像、火焰和声音探测器。
<h1>二、火灾探测的数据融合系统</h1>
<h3>2.1 火灾探测数据融合系统的结构</h3>
本文提出的火灾探测系统有三个级别的融合：信息层、特征层和决策层。信息层主要完成原始数据的采集与处理；特征层利用信息层的输出信号进行数据融合，将火情辨识成明火、阴燃火和非火灾源并判断出大多数场合下的火灾概率；决策层充分利用特征层输出的各类特征信息，采用适当的融合技术和判断规则给出最终火灾结果。火灾探测系统结构如下：

<h3>2.3 特征层实现</h3>
本文提出BP神经网络和模糊逻辑技术来处理火灾的非结构性问题，实现火灾特征层信号的数据融合，并把火灾辨识成明火、阴燃火和非火灾源。
<h6>2.3.1 BP神经网络在火灾探测系统特征层的应用</h6>
BP神经网络特征融合器的输入有：温度（TEM）、烟雾（SMK）、气体（CO）和红外（RED）信号，输出为：明火概率Y1、阴燃火概率Y2和非火灾源概率Y3。

<h6>2.3.2 模糊逻辑在特征层的应用</h6>
1.考虑实际情况，火灾探测信号都有一个门限阈值，以门限阈值为界，将温度、烟气、CO和红外信号模糊化为2级：火情小（PS）、火情大（PB）。

2.建立模糊规则。模糊规则总是以“IF...THEN...”的形式出现。如：“若温度时TEM，并且烟雾是SMKi，CO气体是COi，红外光是REDi，则火情发生的概率为Oi”。本文系统一共建立16条模糊规则。

<h3>2.4 决策层实现</h3>
根据特征层神经网络特征融合器的输出明火概率Y1、阴燃火概率Y2、非火灾源概率Y3与模糊逻辑特征融合器输出的火灾概率O的差异，结合人的经验进行分析判断。判断规则如下：
<ul>
 	<li>当Y1&gt;0.5且O&gt;0.7，判断为明火火灾；</li>
 	<li>当Y1&gt;0.5且O&lt;0.7，进入明火模糊逻辑推理系统进行推理判断；</li>
 	<li>当Y2&gt;0.5且O&gt;0.7，判断为阴燃火灾；</li>
 	<li>当Y2&gt;0.5且O&lt;0.7，进入阴燃火模糊逻辑推理系统进行推理判断；</li>
 	<li>当Y3&gt;0.5且O&lt;0.3，判断为非火灾源干扰；</li>
 	<li>当Y3&gt;0.5且O&gt;0.7，进入非火灾源模糊逻辑推理系统进行推理判断；</li>
</ul>
